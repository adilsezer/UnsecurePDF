# UnsecurePDF
Python script to unsecure all PDF files in a selected folder.

## Requirements
* lxml 4.5.2
* pikepdf 1.18.0
* Pillow 7.2.0

## Running from source
    $ git clone https://github.com/sezerad/UnsecurePDF.git
    $ cd UnsecurePDF
    $ pip install -r requirements.txt
    $ python UnsecurePDF.py
